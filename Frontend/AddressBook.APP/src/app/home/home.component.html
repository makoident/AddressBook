Welcome to Address Book!


  • A simple application to view, search, create, update and delete contacts.
  • Application needs the backend server. Run the server and enter the server url in /environments/environment.ts and /environments/environment.prod.ts .
  • Note: First query to the backend server will take a little longer to execute because the DB will be created on the server on the first query.
  • Author: Marko Sekardi



To use the app, click on a link in the menu on the top of the page.
